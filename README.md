# OFF-quick-add-pwa
Web mobile app to quickly adding data to Open Food Facts

Usage:
* enter account and password
* select a country
* optionally select a category (you have to type 4 letters and wait for off categories to be displayed)

Then you scan a product:
* it verifies if the country and the category already exist
* it updates the product only if the information does not exist yet

The category is very useful if you're playing in a supermarket. If you're facing a shelf with waters, you just select "Waters" as a category, and you can scan dozens of products.
